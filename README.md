# LogAnalyzer

基于 Electron + React + ECharts 的桌面日志分析工具，类似 TextAnalysisTool 加上图表功能。

专为嵌入式/ISP/图像算法工程师设计，用于快速分析调试日志。

## 功能

### 日志查看
- 文件加载（支持大文件流式加载 + 2万行限制）
- 虚拟滚动（支持超大文件流畅浏览）
- 文件拖拽打开
- 暗色主题

### 搜索与高亮
- 关键字搜索（回车自动添加为筛选条件）
- 多关键字同时高亮（不同颜色）
- 支持正则表达式
- 搜索匹配数实时显示

### 筛选
- 过滤/全显双模式
- 包含/排除条件
- 整行高亮标记
- 独立字体颜色
- 拖拽排序
- 批量操作（全选/全不选/清空）
- 筛选方案保存/加载

### 数据图表
- 正则提取数值，自动绘制折线图
- 多指标叠加显示
- 自定义 X 轴（行号或数据字段）
- 阈值线
- 图表导出（PNG/CSV）
- 统计摘要（min/max/avg/n）
- 多分组正则支持

### 日志↔图表联动
- 点击图表自动跳转到对应日志行
- 双向同步

### 注释与书签
- 右键添加行注释
- 书签快速定位
- 注释导出

### 右键菜单
- 添加/编辑注释
- 复制行内容
- 复制行号+内容
- 复制上下文 (±2行)

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| Ctrl+O | 打开文件 |
| Ctrl+F | 聚焦搜索框 |
| Ctrl+G | 跳转到行号 |
| Ctrl+1 | 筛选面板 |
| Ctrl+2 | 图表面板 |
| Ctrl+3 | 注释面板 |
| Ctrl+4 | 配置面板 |
| Ctrl+Shift+F | 切换过滤/全显 |
| Ctrl+Shift+C | 清空筛选条件 |
| Esc | 清除搜索/关闭对话框 |
| F1 | 显示/隐藏帮助 |
| F12 | 开发者工具 |

## 安装与运行

```bash
# 安装依赖
npm install

# 构建
npx webpack --mode production

# 运行
npx electron .
```

## 技术栈

- **Electron** - 桌面应用框架
- **React** - UI 框架
- **ECharts** - 图表库
- **Webpack** - 打包工具
